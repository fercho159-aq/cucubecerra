# Prompts para generacion de imagenes — Cucu Becerra

Prompts listos para copiar y pegar en **Google Gemini (Imagen)**.
Todos los prompts estan en ingles para obtener mejores resultados.

> **Paleta general:** cream, ivory, beige, soft gold, warm white.
> **Estilo fotografico:** Editorial / lifestyle (Martha Stewart Weddings, Vogue Ninos).
> **Iluminacion:** Soft natural light, golden hour or side window light.
> **Regla de personas:** No identifiable faces. Only hands, feet, silhouettes, or overhead angles.

## Notas de uso

- **Formato:** Exportar todas las imagenes en `.webp` para mejor rendimiento web.
- **Compresion:** Calidad 80-85% para fotografias, 90% para iconos y texturas.
- **Fallback:** Generar tambien versiones `.jpg` para navegadores sin soporte WebP.
- **Responsive:** Para el hero y banners, generar versiones adicionales en 768px y 375px de ancho para mobile.
- **Accesibilidad:** Agregar `alt` descriptivos en espanol para cada imagen en el sitio.
